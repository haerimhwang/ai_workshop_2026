# Postgraduate Workshop on AI Applications in Humanities

## About the Workshop

This workshop reflects a shared commitment to fostering a culture of collaboration and knowledge-sharing among the next generation of humanities researchers. Participants will have the 
opportunity to listen to and engage with **Donald Sturgeon**, a distinguished scholar from outside Hong Kong, whose plenary address — *"Navigating the AI Frontier in Humanities Research"* — will set the 
intellectual tone for the event.


##  Program
- 📖 **Use of C-Text**: Exploring classical Chinese text databases
- 🐍 **Web Scraping in Python**: Automated data collection from YouTube comments
- 🖼️ **Image Data Analysis and Machine Learning in Orange**: Clustering historical/cultural image data
- 💬 **Sentiment Analysis in Python**: Analyzing sentiment of YouTube comments
- 🔗 **Keyword and Semantic Network Analysis*: Analyzing kewords and semantic networks using psycholinguistic and historical data
